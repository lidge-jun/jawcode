SDK Overview

The Jawcode SDK (jawcode/sdk) is the in-process integration surface for embedding the agent in your own Bun or Node process. It provides direct access to agent state, event streaming, tool wiring, and session control.

For cross-language or process-isolated scenarios, use RPC mode instead.

Installation

bun add @gajae-code/coding-agent

The SDK is exported from the package root and also via @gajae-code/coding-agent/sdk.

Core Exports

  • createAgentSession — main entry point for creating agent sessions
  • SessionManager — file-backed or in-memory session management
  • Settings — configuration management
  • AuthStorage — credential storage abstraction
  • ModelRegistry — model discovery and selection
  • discoverAuthStorage — auto-discovery of auth configuration
  • discoverContextFiles, discoverPromptTemplates — discovery helpers
  • createTools, BUILTIN_TOOLS — tool factory surface

Quick Start

import { createAgentSession } from "@gajae-code/coding-agent";

const { session, modelFallbackMessage } = await createAgentSession();

if (modelFallbackMessage) {
  process.stderr.write(`${modelFallbackMessage}\n`);
}

const unsubscribe = session.subscribe((event) => {
  if (
    event.type === "message_update" &&
    event.assistantMessageEvent.type === "text_delta"
  ) {
    process.stdout.write(event.assistantMessageEvent.delta);
  }
});

await session.prompt("Summarize this repository in 3 bullets.");
unsubscribe();
await session.dispose();

Session Management

File-Backed (Default)

import { createAgentSession, SessionManager } from "@gajae-code/coding-agent";

const { session } = await createAgentSession({
  sessionManager: SessionManager.create(process.cwd()),
});

console.log(session.sessionFile); // absolute .jsonl path

Persists conversation, messages, and state deltas to session files. Supports resume, open, list, and fork workflows.

In-Memory

const { session } = await createAgentSession({
  sessionManager: SessionManager.inMemory(),
});

console.log(session.sessionFile); // undefined

No filesystem persistence. Useful for tests, ephemeral workers, and request-scoped agents.

Resume and List

import { SessionManager } from "@gajae-code/coding-agent";

const recent = await SessionManager.continueRecent(process.cwd());
const listed = await SessionManager.list(process.cwd());
const opened = listed[0]
  ? await SessionManager.open(listed[0].path)
  : null;

Model and Auth Wiring

import {
  createAgentSession,
  discoverAuthStorage,
  ModelRegistry,
  SessionManager,
} from "@gajae-code/coding-agent";

const authStorage = await discoverAuthStorage();
const modelRegistry = new ModelRegistry(authStorage);
await modelRegistry.refresh();

const available = modelRegistry.getAvailable();
if (available.length === 0)
  throw new Error("No authenticated models available");

const { session } = await createAgentSession({
  authStorage,
  modelRegistry,
  model: available[0],
  thinkingLevel: "medium",
  sessionManager: SessionManager.inMemory(),
});

Model Selection Order

When no explicit model or modelPattern is provided:

  1. Restore model from existing session (if restorable and key available)
  2. Settings default model role (default)
  3. First available model with valid auth

If restore fails, modelFallbackMessage explains the fallback.

Event Subscription

const unsubscribe = session.subscribe((event) => {
  switch (event.type) {
    case "agent_start":
    case "turn_start":
    case "tool_execution_start":
      break;
    case "message_update":
      if (event.assistantMessageEvent.type === "text_delta") {
        process.stdout.write(event.assistantMessageEvent.delta);
      }
      break;
  }
});

Event Types

Core AgentEvent plus session-level events:

  • auto_compaction_start / auto_compaction_end
  • auto_retry_start / auto_retry_end
  • retry_fallback_applied / retry_fallback_succeeded
  • ttsr_triggered
  • todo_reminder / todo_auto_clear
  • irc_message

Prompt Lifecycle

session.prompt(text, options?) is the primary entry point:

  1. Optional command/template expansion (/ commands, file slash commands, prompt templates)
  2. If currently streaming: requires streamingBehavior: "steer" | "followUp"
  3. If idle: validates model + API key, appends user message, starts agent turn

Related APIs

Startup Performance

createAgentSession() runs two background optimizations:

Model-Host Preconnect

As soon as the model is resolved, a best-effort fetch.preconnect(model.baseUrl) fires so DNS + TCP + TLS + HTTP/2 happen in parallel with tool registry build and system-prompt assembly. Saves 100-300ms on transcontinental hops. Silently skipped on non-Bun runtimes.

Conditional LSP Warmup

Startup LSP servers are only warmed when all conditions hold:

  • enableLsp !== false
  • hasUI === true (interactive TUI)
  • lsp.diagnosticsOnWrite is enabled

Print, script, RPC, and ACP invocations skip warmup entirely. Tools that need LSP still spin one up on demand via getOrCreateClient().

Minimal Controlled Example

import {
  createAgentSession,
  discoverAuthStorage,
  ModelRegistry,
  SessionManager,
  Settings,
} from "@gajae-code/coding-agent";

const authStorage = await discoverAuthStorage();
const modelRegistry = new ModelRegistry(authStorage);
await modelRegistry.refresh();

const settings = Settings.isolated({
  "compaction.enabled": true,
  "retry.enabled": true,
});

const { session } = await createAgentSession({
  authStorage,
  modelRegistry,
  settings,
  sessionManager: SessionManager.inMemory(),
  toolNames: ["read", "grep", "find", "edit", "write"],
  enableLsp: true,
});

session.subscribe((event) => {
  if (
    event.type === "message_update" &&
    event.assistantMessageEvent.type === "text_delta"
  ) {
    process.stdout.write(event.assistantMessageEvent.delta);
  }
});

await session.prompt("Find all TODO comments and propose fixes.");
await session.dispose();

CLI-Jaw Embedding

Jawcode's SDK embedding surface (jawcode/sdk) enables integration with the cli-jaw manager platform. The core IPABCD orchestration loop and HITL/HOTL workflow engine were designed for this embedding scenario.

AspectDetail
Session isolation TUI and Web sessions are not shared. jwc uses ~/.jwc/agent/sessions; cli-jaw Web sessions use jaw.db. Search federation is a future addition.
OAuth sharing AuthStorage injection via SDK. Local token seeding through discoverAuthStorage().
Memory integration Current memory root is ~/.jwc/agent/memories/<encoded-cwd>. Cross-platform memory federation is planned.
