Configuration Reference

All settings are defined in config.yml (located at ~/.jwc/agent/config.yml) or set via the in-TUI settings panel (/settings). Settings are organized into tabs.

Identity

Agent persona rendered into the system prompt identity block. When all fields are unset, no identity block is rendered.

KeyTypeDefaultDescription
identity.namestring—Agent display name
identity.emojistring—Signature emoji
identity.vibestring—Tone and personality (separate with ; or newlines)
identity.languagestring—Preferred response language (e.g. Korean, English)
