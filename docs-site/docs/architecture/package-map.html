Package Map

Jawcode is a Bun monorepo with 8 TypeScript packages, a Rust workspace containing 6 crates, and 2 Python packages. This page documents every package, its role, key exports, and where it sits in the dependency graph.

TypeScript Packages

packages/jwc — jawcode

npm namejawcode
Binaryjwc
SDK exportjawcode/sdk
Key filesbin/jwc.js, src/sdk.ts, src/index.ts
Depends on@gajae-code/coding-agent

The public npm package and CLI entry point. bin/jwc.js is a Node.js launcher that resolves a managed Bun runtime and delegates to the coding-agent CLI. jawcode/sdk re-exports the coding-agent SDK for programmatic embedding — this is the official integration surface for external hosts like cli-jaw.

A Node.js compatibility layer lives at dist-node/ with esbuild-bundled shims for Bun.* APIs, bun:sqlite, and other Bun-specific surfaces. Node 22+ SDK import and createAgentSession() streaming are verified green.

packages/coding-agent — @gajae-code/coding-agent

RoleCLI runtime: tools, session, skills, prompts, modes, workflow
Key filessrc/cli.ts, src/main.ts, src/sdk.ts
Depends onagent-core, ai, tui, natives, utils, stats

The primary product surface. Contains all built-in tools (read, bash, edit, AST, eval, find, search, LSP, browser, task/subagent, IRC, web search, etc.), the default workflow skills (jaw-interview, plan, goal, team), bundled role agent prompts (executor, architect, planner, critic), extensibility surfaces (skills, slash commands, custom tools, MCP), session management, and all execution modes (interactive TUI, print, RPC, ACP, bridge).

packages/agent — @gajae-code/agent-core

RoleProvider-agnostic agent loop, state management, compaction
Key exportsAgent, agentLoop, agentLoopContinue, AppendOnlyContextManager
Depends onai, natives, utils

The core agent runtime. Owns the turn loop (prompt → model call → tool dispatch → result append → continue/yield), mutable agent state, subscriptions, abort/pause APIs, queued message handling, provider session state, telemetry, and context compaction. Does not know about specific tools or the TUI.

packages/ai — @gajae-code/ai

RoleMulti-provider LLM client with streaming
Key exportsstreamSimple, ModelManager, AuthBroker, model registry, provider implementations
Depends onutils, provider SDKs (OpenAI, Anthropic, etc.)

Supports 46+ providers and 3,600+ models. Handles model registry/resolution (static, cached, dynamic, remote sources), provider-specific streaming normalization, auth broker/gateway, OAuth flows, API key management, retry/overflow, usage tracking, tool calling with partial streaming, and thinking/reasoning mode control.

packages/tui — @gajae-code/tui

RoleTerminal UI framework with differential rendering
Key exportsComponents (text, input, editor, markdown, select, image, box), TUI core, keybindings
Depends onnatives, utils

Custom terminal UI library — not a wrapper around blessed/ink. Manages component rendering with diffing, focus management, overlays, terminal dimensions, cursor control, Kitty graphics protocol, synchronized output, and appearance notifications. The welcome banner gradient sweep animation and viewport scroll model (B2-lite fill + sticky gap) are hand-tuned product features.

packages/natives — @gajae-code/natives

RoleRust N-API bindings for performance-critical operations
Key exportsgrep, AST search/edit, clipboard, filesystem scan, syntax highlighting, PTY, shell, token counting, text measurement
Depends oncrates/pi-natives (build artifact)

Platform-specific native binaries are resolved at runtime with version validation between the package and the native binary. Covers: AST search and structural editing (tree-sitter/ast-grep), recursive grep/glob, clipboard access, filesystem scan and cache, syntax highlighting, HTML-to-Markdown conversion, keyboard parsing, process/PTY/shell support, SIXEL image rendering, code summarization, token counting, and text width measurement/wrapping/truncation.

packages/utils — @gajae-code/utils

RoleShared TypeScript utilities
Key exportsLogger, config dirs, color/env helpers, fetch retry, formatting, glob, JSON, MIME, sanitization, process tree, temp files
Depends onnatives, handlebars, winston

Common utilities used across all packages. Includes the centralized logger (required instead of console.log in coding-agent to avoid TUI corruption), abortable/async helpers, frontmatter parsing, prompt rendering via Handlebars, and process tree management through native bindings.

packages/stats — @gajae-code/stats

RoleLocal observability dashboard
Key exportsjwc-stats CLI, aggregation APIs, dashboard server
Depends onai, utils, React, Tailwind, Chart.js

Parses session-derived request metrics, writes aggregated data through SQLite, and serves a local dashboard SPA with API routes. Accessed via jwc stats.
