Tool API Reference

Complete parameter reference for every built-in tool. Sourced from the canonical BUILTIN_TOOLS registry in packages/coding-agent/src/tools/index.ts. Tools marked conditional require a settings flag or runtime state to be available.

Essential tools (always loaded)

These 14 tools are loaded by default in every session unless overridden via tools.essentialOverride:

read, bash, edit, write, find, search, ast_grep, ast_edit,
web_search, browser, ask, subagent, task, skill

Filesystem & Code

read

ParamTypeRequiredDescription
pathstringYesLocal path, internal URI (agent://, artifact://, rule://, local://), or URL. Append :<sel> for line ranges, raw mode, or special modes.

Selectors: :50, :50-200, :50+150, :20+1, :5-16,960-973, :raw, :2-4:raw, :conflicts. Reads files, directories, archives (.tar/.zip), SQLite, images (metadata), documents (PDF/DOCX/PPTX/XLSX/RTF/EPUB/ipynb), and web URLs (reader-mode).

bash

ParamTypeRequiredDescription
commandstringYesShell command to execute
cwdstringNoWorking directory (prefer over cd dir &&)
envobjectNoExtra environment variables
ptybooleanNoRun in PTY mode (for sudo, ssh)
timeoutnumberNoTimeout in seconds (default 300)

Returns output and exit code. Stdout/stderr merged. Truncated output available via artifact://<id>.

edit

ParamTypeRequiredDescription
pathstringYesTarget file path
inputstringYesPatch language input using line anchors and ops

Compact line-anchored edit format with » (insert after), « (insert before), := (replace/delete) ops. Anchors reference lines by number + 2-char content hash.

write

ParamTypeRequiredDescription
pathstringYesFile path (supports archive entries and SQLite row ops)
contentstringYesFile content

Creates or overwrites files. Supports .tar/.zip archive entries (archive.ext:path/inside) and SQLite row operations (db.sqlite:table for insert, db.sqlite:table:key for update/delete).

find

ParamTypeRequiredDescription
pathsstring[]YesGlobs, files, or directories
gitignorebooleanNoRespect .gitignore (default true)
hiddenbooleanNoInclude hidden files (default true)
limitnumberNoMax results (default 1000)
timeoutnumberNoTimeout in seconds (0.5-60, default 5)

Returns matching file paths sorted by mtime (most recent first). Truncated at 1000 entries or 50KB.

search

ParamTypeRequiredDescription
patternstringYesRegex pattern (Rust/RE2 syntax, no lookaround)
pathsstring[]YesFiles, directories, globs, or internal URLs to search
ibooleanNoCase-insensitive search
gitignorebooleanNoRespect .gitignore
skipnumberNoFiles to skip (pagination)

ast_grep

ParamTypeRequiredDescription
patstringYesAST pattern. $NAME=one node, $_=ignore one, $$$NAME=zero-or-more
pathsstring[]YesFiles, directories, globs to search
skipnumberNoMatches to skip

Structural AST matching via native ast-grep. Language inferred from paths. Matches AST structure, not text.

ast_edit

ParamTypeRequiredDescription
opsarrayYesRewrite ops. Each: { pat: string, out: string }. Empty out deletes.
pathsstring[]YesFiles, directories, globs to rewrite

Structural AST-aware rewrites. Metavariables captured in pat are substituted into out.

Agent & Task Management

task

ParamTypeRequiredDescription
agentstringYesRole: executor, executor_ext, planner, architect, critic
tasksarrayYesTasks to parallelize. Each: { id, description, assignment, model?, inheritContext? }
contextstringNoShared background prepended to every assignment
schemastringNoJTD schema for structured output
spawnPlanobjectNoRequired for batches > 4 tasks. Fields: whyParallel, whyNotLocal, independence, expectedReceiptShape, maxInlineTokens

Launches subagents to parallelize workflows. Results delivered automatically. Max 4 tasks without spawnPlan.

subagent

ParamTypeRequiredDescription
actionstringYeslist, inspect, await, pause, resume, steer, cancel
idsstring[]NoTarget subagent ids
idstringNoSingle subagent id (for resume/steer)
messagestringNoMessage for resume/steer
timeout_msnumberNoAwait timeout (0-3600000ms)
verbositystringNoreceipt (280 chars), preview (2000), full (12000, requires ids)
pausebooleanNoPause after steering

Manage detached task subagents. Await timeout does not stop the subagent. Cancel only when actually failed/off-track.

irc

ParamTypeRequiredDescription
opstringYeslist or send
tostringFor sendRecipient id or "all"
messagestringFor sendText message
awaitReplybooleanNoWait for reply (default true for send)

Background & Scheduling

background

ParamTypeRequiredDescription
opstringYeslist, detail, follow, cancel, settings

Manage background rows created by task, bash (async), and monitor. Launch stays with the originating tool. TUI footer rows and this tool's rows are the same surface.

job

ParamTypeRequiredDescription
listbooleanNoEnumerate running jobs
pollstring[]NoBlock until specified jobs finish or timeout
cancelstring[]NoStop running jobs

Legacy async job surface. Prefer background for canonical row management.

monitor

ParamTypeRequiredDescription
commandstringYesShell command to run. Each stdout line becomes an event.
kindstringYeslog, poll, watch, other
descriptionstringYesHuman-readable description
timeoutnumberNoMax seconds before auto-shutdown
persistentbooleanNoKeep running past current turn (default false)

Background event stream. Each stdout line delivered as <task-notification> between turns.
