Themes

Jawcode's TUI appearance is fully driven by theme JSON files. Each theme defines color variables, semantic color mappings, exported page colors, and symbol overrides.

Default Theme: abyss-bite

The default dark theme for Jawcode is abyss-bite — a deep ocean palette with cyan accents and an orange bite highlight. The light counterpart is abyss-bite-light.

Theme JSON Schema

Every theme file follows this structure:

{
  "$schema": "https://raw.githubusercontent.com/.../theme-schema.json",
  "name": "my-theme",
  "vars": {
    "cyan": "#5fd6e0",
    "accent": "#00d1da",
    ...
  },
  "colors": {
    "accent": "cyan",          // reference a var name
    "border": "#14405e",       // or a direct hex value
    "toolTitle": "shell",
    ...
  },
  "export": {
    "pageBg": "#0a0d12",       // exported for external consumers (stats dashboard, etc.)
    "cardBg": "#0e131b",
    "infoBg": "#14202c"
  },
  "symbols": {
    "preset": "unicode",       // "unicode" | "nerd" | "ascii"
    "overrides": {
      "icon.pi": "🦈",
      "md.bullet": "▸"
    }
  }
}

Section Reference

SectionPurpose
varsRaw color definitions. Referenced by name in the colors section. Theme consumers resolve var references recursively.
colorsSemantic color mappings. Keys like accent, border, toolTitle, syntaxKeyword map to either a var name or a hex literal. This is what the renderer reads at runtime.
exportPage-level colors for external consumers (stats dashboard, web embeds). Not used by the TUI renderer directly.
symbolsSymbol preset selection and per-symbol overrides. preset chooses the base symbol set; overrides replaces individual symbols by key.

Welcome Banner

The welcome banner is a hand-tuned visual feature, not incidental decoration. It displays the JAW_FIN_LOGO ASCII art with a multi-stop gradient sweep animation:

  • Gradient stops from JAW_GRADIENT_STOPS: abyss navy → deep teal → cyan glow → bright cyan → bite orange tip
  • INTRO_SWEEPS animation applies the gradient as a moving band across the ASCII art
  • The "Jawcode" wordmark and "bite · build · ship" tagline render below
  • Model/provider pills display the active model and authenticated providers

The banner, sweep animation, and shine effects in welcome.ts must not be simplified, flattened, or removed. They are a protected product feature.

Custom Themes

To create a custom theme:

  1. Create a JSON file following the schema above
  2. Place it in a location accessible to jwc
  3. Set theme.dark or theme.light in your config to the theme name

Custom themes can reference any var names in their colors section, define new vars, and override any symbol key.
