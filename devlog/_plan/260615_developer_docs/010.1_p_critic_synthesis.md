## Per-Page Content Specs (F1)

### Getting Started
- **installation**: npm 1-liner, bun from source, binary download (향후), Node/Bun 요구사항, `jwc --version` 검증
- **quickstart**: `jwc` 실행, 첫 메시지 보내기, `/skill:jaw-interview` 시도, `jwc orchestrate p` 첫 plan, 결과 확인
- **cli-reference**: `jwc --help` 전체 플래그 (args.ts 파싱), `--tmux`, `--worktree`, `--model`, `--mode`, env overrides
- **environment-variables**: `JWC_CONFIG_DIR`, `ANTHROPIC_API_KEY`, `OPENAI_API_KEY` 등 코드에서 grep한 전체 목록 + 설명
- **troubleshooting**: Bun 버전 호환, API key 인증 실패, context overflow, theme 로드 실패, model 미발견

### Architecture
- **overview**: 8 packages + 1 crate 다이어그램, 패키지 간 의존 그래프, entry points
- **package-map**: 패키지별 책임/크기/주요 export 테이블
- **agent-loop**: turn cycle 플로우 (user msg → tool dispatch → streaming → yield), tool choice queue
- **prompt-assembly**: Handlebars 템플릿 → system prompt 배열 → API, section 순서 테이블
- **compaction**: auto-compact threshold 계산, microcompact, continuation summary 포맷
- **binary-build**: bun build → gjc 바이너리, worker entrypoints, sidecar compile
