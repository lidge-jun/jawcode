## 목표

Node 22 `dist-node/sdk.js` 경로에서 실제 provider 1개로 스트리밍 왕복을 완료한다. 이것이 100 MOC의 핵심 완료 기준이다.

## 범위

### 포함

- Claude 또는 OpenAI 중 현재 로컬 credential이 준비된 provider 1개 선택
- `createAgentSession()` 생성
- `prompt("Say hello in one short sentence")` 실행
- streaming delta/final 확인
- auth storage가 Node sqlite adapter로 동작하는지 확인

### 제외

- 모든 provider matrix
- cli-jaw 서버 attach
- tool execution full e2e
- long context/resume 검증

## provider 선택 규칙

1. 로컬 credential이 이미 존재하는 provider 우선
2. 새 토큰 입력/로그인은 이 문서 범위 밖
3. credential이 없으면 100.11은 blocked로 기록하고 100.12 closeout에서 미완료 사유로 남김

## 검증 스크립트 요구

출력해야 하는 증거:

- provider name/model
- delta count
- final text 일부
- 종료 코드 0
- 사용한 agentDir/cwd가 임시 경로인지 여부

민감정보 금지:

- token 원문
- auth DB path의 credential payload
- full request/response body

## 실패 분류

| 실패 | 처리 |
|------|------|
| auth DB load 실패 | 100.05 sqlite adapter 보정 |
| provider SDK import 실패 | build external/dependency 보정 |
| stream parsing 실패 | 100.05 JSONL 보정 |
| prompt/system load 실패 | 100.06 보정 |
| network/provider auth 실패 | 코드 문제가 아니면 blocked 기록 |
