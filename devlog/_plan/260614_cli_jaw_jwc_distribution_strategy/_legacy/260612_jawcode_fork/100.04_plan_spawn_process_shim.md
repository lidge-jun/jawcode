## 구현 설계

Node 대응은 `node:child_process.spawn`를 사용한다.

필요 최소 반환 객체:

```ts
{
  pid,
  stdout,
  stderr,
  stdin,
  exited: Promise<number>,
  kill(signal?),
}
```

Bun과 Node의 차이:

| Bun | Node | 처리 |
|-----|------|------|
| command array | command + args | 첫 원소 분리 |
| `stdout: "pipe"` | `stdio` 배열 | 옵션 변환 |
| `stderr: "pipe"` | `stdio` 배열 | 옵션 변환 |
| `stdin: "pipe"` | `stdio` 배열 | 옵션 변환 |
| Web stream | Node stream | 필요 시 `Readable.toWeb` |
