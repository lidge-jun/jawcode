- TUI-only Bun API (bun:ffi) — lazy-throw 별칭, 호출부 try/catch (Node SDK graph 무관)

## deferred 허용 목록

100 closeout에서 deferred로 남길 수 있는 항목:

| 항목 | 조건 |
|------|------|
| TUI-only Bun API | Node SDK graph에 포함되지 않음 |
| STT/pty 경로 | M2 resident engine 필수 경로 아님 |
| auth-broker/gateway full server | 실 provider SDK hello world에 불필요 |
| archive write edge cases | read/write 핵심 도구가 Node import를 막지 않음 |

허용되지 않는 deferred:

- `createAgentSession` import 실패
- provider credential load 실패가 코드 원인인 경우
- stream parser 실패
- prompt/defaults 로드 실패
- Bun TUI 경로 회귀

## 110 handoff 계약

```text
import { createAgentSession } from "jwc/sdk"  // cli-jaw에서 단일 import
```

110 JawRuntime 배선이 엔진 SDK 표면을 건드렸으므로(`sdk.ts`에 `SessionManager`/`AgentSession(Event)` 재수출 추가) 100 회귀를 재검증:

|---|---|
| `bun run build:node` | green — `dist-node/sdk.js` 재생성, exports **25→26** (`SessionManager` 추가분) |

| `smoke-node-streaming.mjs` (100.10) | PASS — Node에서 7개 이벤트(agent_start…agent_end) 발화 + text round-trip |

**결론**: 100 무회귀. 추가로, 110 매퍼/JawRuntime가 의존하는 `AgentSessionEvent` 스트림이 Node 경로에서 정상 발화함이 입증됨(110.4 표의 전제 충족). `dist-node/`는 gitignored build artifact이므로 cli-jaw는 `JWC_SDK_PATH=<repo>/packages/jwc/dist-node/sdk.js`로 dev 로드, 정식 활성은 npm link 후(150.1 P-2).
