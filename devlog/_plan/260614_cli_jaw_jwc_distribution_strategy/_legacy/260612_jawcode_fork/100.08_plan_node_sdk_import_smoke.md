## 목표

`packages/jwc/dist-node/sdk.js`를 Node 22에서 import하고, 공개 SDK export 중 `createAgentSession`이 존재함을 확인한다.

## 범위

### 포함

- `bun --cwd=packages/jwc run build:node`
- `node -e "import('./packages/jwc/dist-node/sdk.js')..."`
- export key 확인
- import 중 side-effect 오류 확인
- 실패 시 stack trace를 API 카테고리로 분류

### 제외

- 실제 LLM provider 호출
- 스트리밍 완주
- cli-jaw 서버 attach

## smoke script

기본 커맨드:

```bash
node -e "import('./packages/jwc/dist-node/sdk.js').then(m => { console.log(Object.keys(m)); if (!m.createAgentSession) throw new Error('missing createAgentSession'); })"
```

권장: `packages/jwc/scripts/smoke-node-sdk.ts`를 추가해도 된다. 단 100.08에서는 일회성 커맨드만으로 충분하다.
