## 1. 슬라이스 (098 PR 번호 매핑)

### A (=110.1 PR) — `jwc-runtime.ts` 뼈대

신규 `cli-jaw/src/agent/jwc-runtime.ts`:
```typescript
import { createAgentSession, type AgentSession } from 'jwc/sdk';  // 100.12 계약: 단일 import
class JawRuntime {
  #session: AgentSession | null = null;
  #inFlight: Promise<{text: string; code: number}> | null = null;
  async init(cwd: string) {            // createAgentSession({ cwd, agentDir: JWC_AGENT_DIR, hasUI: false, sessionManager })
  async prompt(text): SpawnResult-호환  // #inFlight 래핑
  async steer(text)                     // isStreaming ? prompt(text,{streamingBehavior:"steer"}) : prompt(text)
  async followUp(text)
  subscribe(listener)
  get busy() { return !!this.#inFlight || (this.#session?.isStreaming ?? false); }
  async dispose() / async #recover(cwd) // dispose→null→init 재생성, 서버 생존
}
```

**완료**: mock session 단위 테스트 — init/prompt/busy/dispose lifecycle.
