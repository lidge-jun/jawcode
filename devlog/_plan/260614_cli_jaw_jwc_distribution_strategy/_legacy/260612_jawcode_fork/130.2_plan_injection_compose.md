### Slice B — auth 시딩 브리지 (M2 done ②) — **110과 병행 가능**

cli-jaw 서버 기동 경로에 추가:
```typescript
const storage = await discoverAuthStorage("~/.jwc/agent");   // sdk.ts:409
if (!storage.has("anthropic")) {
  const local = detectClaudeCodeToken();                      // Keychain "Claude Code-credentials"
  if (local) await storage.set("anthropic", { type: "oauth", ...local });
}
```
대안: cli-jaw가 auth-broker 서버 역할 → jwc에 `GJC_AUTH_BROKER_URL` 주입 (브로커 모드).
**검증**: 신규 머신 시나리오 — Claude 로그인만으로 Web UI 1턴 (`auth-broker-import.test.ts` green 유지 + 기동 시딩 신규 테스트).
