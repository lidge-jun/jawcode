| 열 | 정의 |
|---|---|
| 기능 | 벤더 CLI 기능 단위 |
| jwc 충당 | 동등 도구/커맨드 (위 전수 목록 기준) |
| lib/mcp 충당 | cli-jaw 측 보완 경로 |
