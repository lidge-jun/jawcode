# 99.20.00 MOC — 프런트 UI/UX 전담 밴드

> 상태: 🟡 밴드 개설 (260612 밤) — 스캐폴딩 ✅ / 첫 슬라이스 선정 대기.

> 대역인 것과 구분해 **99.2x를 프런트 UI/UX 신규 작업 대역**으로 예약한다.

jwc의 TUI 표면은 080~086 밴드에서 큰 골격(자동접힘·포커스 링·오버레이·비주얼 아이덴티티·
인터뷰 v2 입력 슬롯)이 이미 완성됐다 — 260612 밤 코드 검증으로 확인 (아래 §기준선). 따라서
본 밴드는 잔여 추적이 아니라 **앞으로의 프런트 UI/UX 신규 슬라이스가 등재되는 수렴지**다.
레이아웃·컴포넌트·인터랙션·비주얼 등 TUI 표면을 건드리는 새 작업은 흩어지지 말고 여기
`99.20.NN`으로 들어온다.
