## 원칙

- **커밋 단위 = 표의 P 행** (원자·독립 롤백 가능). β의 P9+P10은 연속 필수(중간 상태에서 CI red 허용 구간 — 같은 PR/세션 내).
- 매 P 후: `bun run check:ts` + 해당 게이트 + `Fork-Delta:` 트레일러(P1 이후 규약 적용).
- 각 P는 fork-delta.md 동행 갱신 (P1에서 체계 선행하는 이유).
- 검증 티어: P2~P7 STANDARD / P9~P11 THOROUGH (>20파일) / P12 THOROUGH(release 표면).
