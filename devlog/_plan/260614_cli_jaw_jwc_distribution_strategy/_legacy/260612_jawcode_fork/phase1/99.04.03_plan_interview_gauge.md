## 3. 구현 diff

### M1. `skill-state/workflow-hud.ts` — `buildJawInterviewHudSummary()` 확장

```typescript
// state.state.rounds[last]에 dimension 점수가 있으면:
const dims = lastRound?.dimensions; // { goal, constraint, success, ontology } 0..3 스케일 가정 — 실제 키는 040 상태 스키마 확인
if (dims) {
	details.push({
		label: "dims",
		value: (["goal", "constraint", "success", "ontology"] as const)
			.map(k => `${k[0].toUpperCase()}${gaugeBar(dims[k])}`)
			.join(" "),
		priority: 40,
	});
}
const prev = rounds.at(-2)?.current_ambiguity;
if (typeof prev === "number" && typeof ambiguity === "number") {
	const delta = ambiguity - prev;
	chips.push({ label: "Δ", value: `${delta <= 0 ? "↓" : "↑"}${Math.abs(delta).toFixed(2)}`,
		severity: delta <= 0 ? "success" : "warning", priority: 35 });
}
```

`gaugeBar(n)` = `"▰".repeat(n) + "▱".repeat(3-n)` — workflow-hud.ts 내 로컬 헬퍼.
