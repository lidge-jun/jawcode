|---|---|---|

## 구현

### 감지 함수

```typescript
// in local-token-detect.ts (or inline in openai-codex.ts)
import { existsSync, readFileSync } from "node:fs";
import { join } from "node:path";
import { homedir } from "node:os";

function getCodexHome(): string {
  return process.env.CODEX_HOME ?? join(homedir(), ".codex");
}

export function detectCodexCliToken(): OAuthCredentials | null {
  const authPath = join(getCodexHome(), "auth.json");
  if (!existsSync(authPath)) return null;

  try {
    const raw = JSON.parse(readFileSync(authPath, "utf8"));
    const tokens = raw?.tokens;
    if (!tokens?.access_token || !tokens?.refresh_token) return null;

    // JWT exp 추출 (access_token의 payload.exp)
    const payload = JSON.parse(
      Buffer.from(tokens.access_token.split(".")[1], "base64url").toString()
    );
    const expiresMs = (payload.exp ?? 0) * 1000;

    return {
      refresh: tokens.refresh_token,
      access: tokens.access_token,
      expires: expiresMs,
      accountId: tokens.account_id,
    };
  } catch {
    return null;
  }
}
```

### login 함수 수정 (`openai-codex.ts`)

```typescript
export async function loginOpenAICodex(ctrl: OAuthController): Promise<OAuthCredentials> {
  // Auto-detect: Codex CLI 로컬 토큰
  const local = detectCodexCliToken();
  if (local) {
    ctrl.log?.("Found Codex CLI token, importing automatically");
    if (local.expires < Date.now() + 60_000) {
      try { return await refreshOpenAICodexToken(local.refresh); }
      catch { /* refresh 실패 → 기존 플로우 */ }
    } else {
      return local;
    }
  }

  // 기존 browser OAuth 플로우 (변경 없음)
  ...
}
```

## 엣지 케이스

| 상황 | 처리 |
|---|---|
| `~/.codex/` 디렉토리 없음 | `null` → 기존 플로우 |
| JSON 파싱 실패 | `null` → 기존 플로우 |
| 토큰 만료 + refresh 성공 | 새 토큰 반환 |
| 토큰 만료 + refresh 실패 | `null` → 기존 플로우 |
| 토큰 유효 | 바로 반환 |
| 다른 OpenAI 계정 | accountId 포함하여 저장 (jwc가 계정 구분) |
| Codex CLI 실행 중 (파일 lock) | `readFileSync`는 lock 무시 — 문제 없음 |
