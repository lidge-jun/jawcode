## 1. 문제 (배경)

인터뷰(`jaw-interview` → `ask` 도구)에서 TUI는 선택지 + 자유 입력이 필요하다.

| 단계 | 화면 | 불만 |
|------|------|------|
| 레거시 | `Other` 행 → Enter → `> ` 에디터 | 2단계 |
| v1 (코드) | 1…N 옵션 + **아래** 상시 `> ` (`customInputDocked`, Tab) | **거부** (§8) |

근거 코드: `ask.ts` (`useDockedCustomInput`), `hook-selector.ts` (`#mountDockedInputArea()`).

**v2 핵심:** **마지막 슬롯 = 리스트 안 composer-style 입력**, ↑↓ 단일 포커스 루프. 아래 도킹 **없음**.

## 2. v1 목표 UX (폐기 — 참고만)

### 9.2 입력 슬롯 Enter (skip)

```ts
const text = editor.getText().trim();
if (text.length === 0) {
  // skip: no onSubmit, no customInput, no auto-inject

}
onSubmit(text); // QuestionResult.customInput = text only
```

- **자동 주입 금지:** `""`, `" "`, `"(skipped)"`, 이전 답, SKILL 기본값 등 **일체 없음**.

| autocomplete UI | **허용** — 삽입만 |
