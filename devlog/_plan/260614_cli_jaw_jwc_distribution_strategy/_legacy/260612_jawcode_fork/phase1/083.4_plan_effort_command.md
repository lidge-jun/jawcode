# 083.4 — plan: `/effort` 슬래시 커맨드 (추론 강도 표준 문법)

> 상태: ✅ 구현 완료 (2026-06-12). 소속: 083 ([083_moc_tui_output.md](./083_moc_tui_output.md)).
> 입력: 사용자 "`/effort`가 표준 문법이니까 그거로 구현. claude code나 codex 리포 참고".

jwc에는 추론 강도(ThinkingLevel) 시스템이 있지만 노출 표면이 shift+tab 순환과 `/model sonnet:high`
콜론 문법뿐 — 직접 지정하는 슬래시 커맨드가 없다. 업계 표준 어휘인 **effort**로 커맨드를 추가한다.

## 레퍼런스 (2026-06-12, `~/Developer/codex` 실소스)

- **Codex(Rust)**: `ReasoningEffort` enum = `None/Minimal/Low/Medium/High/XHigh` (기본 Medium,
  `protocol/src/openai_models.rs:44-52`). 노출은 `/model` 피커("choose what model and reasoning effort",
  `tui/src/slash_command.rs:107`) + `model_reasoning_effort` config.
- **Claude Code**: 독립 `/effort` 커맨드 없음 (thinking 토글/budget 계열). → "effort"는 OpenAI/Codex 계열
  표준 어휘. jwc는 **독립 `/effort` 커맨드**로 한 단계 더 직접적으로 노출.

## 매핑 (Codex 표준 ↔ jwc ThinkingLevel)

| 표준(Codex) | jwc 레벨 | 비고 |
|------------|---------|------|
| none | off | 별칭 처리 |
| minimal | min | 별칭 처리 (~1k tokens) |
| low/medium/high/xhigh | 동일 | medium ~8k, high ~16k, xhigh ~32k |
| — | max | jwc 고유 상위 레벨 (유지) |

## 설계

- `builtin-registry.ts`에 `/effort` 추가 — `/fast` 패턴(`:410`) 재사용:
  - **인자 없음 → 드롭다운 셀렉터** (`showEffortSelector()`, selector-controller — `showSelector` +
    SelectList, 현재 레벨 프리선택, Enter 적용/Esc 취소). `status` → 현재 레벨 텍스트 출력.
  - `/effort <level>` → 별칭 정규화(none→off, minimal→min) 후 `parseThinkingLevel()` →
    `session.setThinkingLevel()` (모델별 클램프는 세션/ai 레이어 `clampThinkingLevelForModel`이 처리).
  - 서브커맨드 자동완성: off/minimal/low/medium/high/xhigh/max (+status), 설명은 `thinking.ts` 메타데이터.
- 기존 표면(shift+tab 순환, `/model :level` 문법)은 그대로 — `/effort`는 추가 표면.

## 검증

- ✅ 슬래시 커맨드 스위트 67 pass (cli-command-surface, slash-command-format, acp-builtins, provider) + tsc 클린.
- ✅ 구현: `builtin-registry.ts` `/effort` (handle=ACP/generic + handleTui), `parseEffortArg()` —
  none→off, minimal→min 별칭, inherit 거부. **인자 없는 `/effort` = 드롭다운**(`selector-controller.ts`
  `showEffortSelector()`, types/interactive-mode에 ctx 노출) — ACP/generic 경로는 텍스트 상태 출력 유지.
- ⬜ e2e: `/effort high` → 상태줄/다음 턴 추론 예산 반영, `/effort` → 현재 레벨 표시.
