1. **표시 갭**: `prompts/*.md`는 `with {type:"text"}` 빌드타임 임베드인데 `buildSystemPrompt`의 `data`(`system-prompt.ts:593-619`)에 `APP_NAME`이 없음 — jwc 실행에서도 "You are GJC, the Gajae Code coding agent"(`system-prompt.md:3`)가 그대로 나간다. → **하드 수정으로 해소 (M2)**

3. ~~스킬 본문 직접 수정 불가~~ → **개정**: `default-gjc-definitions.test.ts:343-371`·rebrand G002가 소스 수정을 막고 있었으나, 인터뷰 02:04 확정으로 **가드 자체를 jwc 어휘 기준으로 재작성(M5)** — 소스 수정 차단 해제.
