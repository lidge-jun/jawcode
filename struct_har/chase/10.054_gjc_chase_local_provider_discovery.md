## Why This Is A Gap

GJC added a self-hosted, OpenAI-compatible local provider path: model discovery, per-role model resolution, runtime diagnostics, and graceful fallback when the local endpoint is unavailable. JWC needs to decide whether self-hosted LLMs are a product goal and, if so, whether this discovery/fallback contract fits `.jwc` model config.

Behavioral surface:
1. Local provider model discovery against an OpenAI-compatible endpoint.
2. Diagnostics surface so a misconfigured local endpoint reports clearly.
3. Role-model resolution for local providers (which local model serves which role).
4. Fallback when a configured local provider is unreachable.
5. Resilience when the model registry is missing in the `model` slash command.
