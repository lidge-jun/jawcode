## 변경 내용

Claude Fable/Mythos 모델이 forced tool_choice를 거부(400 에러)하는 문제에 대한 2단 방어:

1. **정적 감지** (`model-thinking.ts` + `models.ts` + `openai-compat.ts`):
   - `isClaudeForcedToolChoiceIncapableModelId(model.id)` → model.compat에 `toolChoiceSupport: "auto"` 설정
   - 번들 카탈로그 로드 시 `applyBundledCompatDefaults`로 stale models.json에도 적용
   - models.dev 동적 디스커버리에도 `anthropicToolChoiceCompat` 적용

2. **런타임 fallback** (`agent-loop.ts` + `agent.ts`):
   - 새 stream event type `"toolChoiceIncapability"` 처리
   - `onToolChoiceIncapability` 콜백을 Agent 옵션으로 전파
   - 모델이 400으로 tool_choice를 거부하면 런타임에서 감격 표시 처리
